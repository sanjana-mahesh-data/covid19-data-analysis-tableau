# COVID-19 Data Analysis Dashboard

## 📊 Project Overview
Analyzed global COVID-19 data to track cases, deaths, and recovery trends.

## 🎯 Objective
- Monitor global COVID trends
- Analyze country-wise case distribution
- Identify patterns in infection rates

## 🛠 Tools & Technologies
- Tableau

## 📂 Dataset
- Global COVID dataset including cases, deaths, and recoveries

## 🔍 Key Insights
- Identified peak infection periods
- High variation across countries
- Trends show waves of infection over time

## 📊 Dashboard Features
- Global map visualization
- Time-series trend analysis
- Country-wise comparison

## 🚀 Outcome
Provided clear visualization of pandemic trends for better understanding and analysis.
